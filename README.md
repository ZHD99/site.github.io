## 特征

- 默认开启Pjax
- 全文搜索
- 调试功能
- 自定义作者
- 自定义二级目录
- 自定义社交按钮
- 自定义音乐播放器
- 自定义悬浮文章上变色
- 自定义外部链接拦截提示
- 支持 valine 评论系统
- 支持config配置友情链接
- 自定义背景设置、自定义作者、底部时间统计等等...
